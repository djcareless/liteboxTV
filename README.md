# LiteBox TV

A fast, lightweight media center app built for Android TV devices (Firestick, NVIDIA Shield, etc.).

Inspired by Kodi and Plex — rebuilt clean for low-power Android TV hardware.

## Features
- Plugin system (JSON manifests)
- Skin/theme system (JSON packs)
- Media3/ExoPlayer playback
- Local + network media sources
- Remote-first navigation
- Firestick optimized

## Tech Stack
- Kotlin
- Jetpack Compose for TV (Leanback)
- Media3 / ExoPlayer
- Room / SQLite
- Gradle (Kotlin DSL)
